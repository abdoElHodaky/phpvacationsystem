
How to run Vacations management system

Download the zip file from github.
Open the command prompt and write the command php -S localhost:4000 to start the php development server
Extract the zip file and copy the files into C:\Users\xxx\vacations_management_system-main\vacations .
Create a database with the name employees
Import employee.sql file ( given inside the database folder )
Run the script file http://localhost:4000/vacations_management_system-main/vacations/index.php
